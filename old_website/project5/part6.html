11.2.6 - Color Image Compression using Luminance & Color Difference Coordinates

Here we will use the same color image and extracted 8x8 block obtained from the previous problem, but this time we transform the RGB values to luminance/color difference coordinates (YUV) using different quantization. Instead we will use the JPEG-suggested matrix for Luminance (\(Y\)) and the Color Difference Matrix for the color differences (\(U\) and \(V\)).

We define \(Y\), \(U\) and \(V\) as follows: \begin{eqnarray*} Y &=& 0.299R + 0.587G + 0.114B \\ U &=& B - Y \\ V &=& R - Y \\ \end{eqnarray*} Then we convert back to RGB by defining \begin{eqnarray*} R &=& V + Y \\ B &=& U + Y \\ G &=& \frac{(Y - 0.299R - 0.114B)}{0.587} \\ \end{eqnarray*} and recombine as a color image.
