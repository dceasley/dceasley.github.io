Introduction

The main goal of image compression is storage efficiency. Image compression aims to reduce the size of stored data by weeding out redundant information in order to save space.

For this project, we were tasked to solve the Computer Problems 3-6 from section 11.2 (p. 513-514) in 'Numerical Analysis' by Tim Sauer.

All subsequent problems will perform various types of image compression on the following image of size 400\(\times\)400 pixels:

The interior of the Basilica Superiore di San Francesco d'Assisi, Italy
